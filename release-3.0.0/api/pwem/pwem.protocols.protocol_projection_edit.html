pwem.protocols.protocol_projection_edit module

class pwem.protocols.protocol_projection_edit.ProtProjectionEditor(**kwargs)[source]

Bases: EMProtocol

This protocol edits the projection directions of all the items of a set of particles using a formula. This could be useful for applying geometrical transformation to a set of particles.

Several predefined operation are offered such as * apply the rotation matrix define by a origin vector and a target vector * rotate the projection vector around a given vector by A degrees * convert between icosahedral symmetries * convert between dihedral symmetries * convert between tetrahedral symmetries
