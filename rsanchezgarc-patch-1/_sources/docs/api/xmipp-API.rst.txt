Xmipp API
===========

XMIPP is not part of Scipion, but Scipion takes some functionality from
`XMIPP <http://xmipp.i2pc.es/>`_. For that reason, if you intended to
develop Scipion or a Scipion's plugin, Xmipp API documentation might be
useful.
